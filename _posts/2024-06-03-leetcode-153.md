## 내가 작성한 풀이

생각보다 쉽게 해결되었다.

문제에서는 다음과 같은 예시들을 제공해준다.

- `[3,4,5,1,2]`
- `[4,5,6,7,0,1,2]`
- `[11,13,15,17]`

여기서 rotate 가 되었다면 중간에 숫자가 작아지는 지점이 생긴다. 만약 rotate 가 되지 않았다면 맨 앞자리가 최소의 수이다.

이 규칙을 수식으로 옮기면 된다.

```java
class Solution {
    public int findMin(int[] nums) {
        int p = 0;
        int min = nums[0];
        while(true) {
            if (p == nums.length) {
                break;
            }

            if (nums[p] > nums[p + 1]) {
                min = nums[p + 1];
                break;
            }

            p++;
        }

        return min;
    }
}
```

### TC, SC

시간 복잡도는 O(n)이고, 공간 복잡도는 O(1)이다.
