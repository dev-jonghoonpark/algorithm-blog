## 내가 작성한 풀이

두 리스트에서 작은 값을 찾고 작은 값을 새 ListNode에 추가한 뒤 기존의 리스트에서는 지워준다.

```java
class Solution {
    public ListNode mergeTwoLists(ListNode list1, ListNode list2) {
        ListNode head = null;
        ListNode tail = null;

        while(!(list1 == null && list2 == null)) {
            ListNode selected;
            if (list1 == null) {
                selected = list2;
                list2 = list2.next;
            } else if (list2 == null) {
                selected = list1;
                list1 = list1.next;
            } else if (list1.val < list2.val) {
                selected = list1;
                list1 = list1.next;
            } else {
                selected = list2;
                list2 = list2.next;
            }

            ListNode newNode = new ListNode(selected.val);
            if(head == null) {
                head = newNode;
            } else {
                tail.next = newNode;
            }

            tail = newNode;
        }

        return head;
    }
}
```

### TC, SC

시간 복잡도는 O(n)이고, 공간 복잡도는 O(n)이다.
