[104. Maximum Depth of Binary Tree](https://leetcode.com/problems/maximum-depth-of-binary-tree/description/)

트리 구조와 관련된 문제이다. DFS를 통해서 풀어야겠다고 느낌이 왔기 때문에 그렇게 진행해보았다.

```py
from typing import Optional


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        if root is None:
            return 0

        def getDepthUsingDFS(node):
            if node is None:
                return 0

            depth = 0
            if node.left is None and node.right is not None:
                depth = getDepthUsingDFS(node.right)
            if node.left is not None and root.right is None:
                depth = getDepthUsingDFS(node.left)
            if node.left is None and node.right is None:
                depth = 0
            else:
                depth = max(getDepthUsingDFS(node.left),
                            getDepthUsingDFS(node.right))
            return depth + 1

        return max(getDepthUsingDFS(root.left), getDepthUsingDFS(root.right)) + 1
```

대부분의 케이스 통과하였으나 아래의 케이스에서 시간 초과로 실패했다.

어떻게 시간을 줄여줄 수 있을까 고민하다보니 조건문들이 불필요하단 것이 보였고 불필요한 조건문을 제거해줘봤다.  
그랬더니 통과했다.

조건문을 빼주는 것으로도 runtime에 유의미하게 영향을 미치는구나 싶었다.

최종 코드는 다음과 같다.

## 최종 코드

```py
from typing import Optional


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        if root is None:
            return 0

        def getDepthUsingDFS(node):
            if node is None:
                return 0

            depth = max(getDepthUsingDFS(node.left),
                        getDepthUsingDFS(node.right))

            return depth + 1

        return max(getDepthUsingDFS(root.left), getDepthUsingDFS(root.right)) + 1
```

## 다른 사람들이 푼 방식

```py
class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        if root is None:
            return 0

        leftMax = 1 + self.maxDepth(root.left)
        rightMax =  1 + self.maxDepth(root.right)

        return max(leftMax, rightMax)
```

별도의 함수를 만들지 않고 문제에서 주어진 함수를 그대로 응용해서 풀었다.  
생각도 못하고 있던 접근이라 "오..." 라는 생각이 들었다.
